## 一、交易品种概览

不同品种在回测和实盘中的数据处理方式差异很大，务必先搞清楚你操作的是什么。

### 1. 股票

- 代表公司所有权的一部分，T+1 交割（A 股），T+0 在美股和港股是主流。
- **量化注意事项**：
  - 除权除息会导致价格跳空，回测时必需复权处理
  - A 股有涨跌停（±10% 普通股票，±20% 科创板/创业板），会影响止损策略的执行
  - 停牌数据需要特殊标记，不可简单填充

### 2. 期货

- 标准化合约，约定未来某个时间以特定价格买卖标的物。T+0，可做多也可做空。
- **量化注意事项**：
  - **换月移仓**：合约有到期日，临近交割必须换到下一主力合约，换月数据拼接时会产生跳空
  - **保证金交易**（杠杆），回测中必须计算占用的保证金，不能全仓
  - 夜盘的存在使得 K 线数据的日期归属需要特殊处理（如 15:00 收盘后，夜盘 21:00 开始算新一天）

### 3. 期权

- 赋予买方在未来以指定价格买卖标的的权利（非义务）。分为看涨（Call）和看跌（Put）。
- **量化注意事项**：
  - 价格受标的价格、行权价、到期时间、波动率等多因素影响，非线性
  - 回测需要考虑隐含波动率曲面，数据量远大于股票
  - 初学者建议先从股票或 ETF 策略入手，期权作为后续方向

### 4. ETF

- 交易型开放式指数基金，本质是一篮子股票，在交易所像股票一样交易。
- **量化注意事项**：
  - 对新手来说，ETF 是比个股更好的策略测试标的（分散风险、数据稳定）
  - 存在折价/溢价（价格与净值不一致），套利机会的窗口极短
  - 注意跟踪误差

### 5. 可转债

- 可以按约定条件转换为股票的债券，兼具债性和股性。
- **量化注意事项**：
  - 有转股期、强赎条款、回售条款等，规则复杂
  - 适合做低风险套利策略（如双低轮动）

### 6. 加密货币

- 7×24 交易，无涨跌停，波动极大。数据来源多依赖交易所 API。
- **量化注意事项**：
  - 永续合约的资金费率是关键的策略输入
  - 数据噪声极大，需要更严格的数据清洗
  - 交易所宕机、拔网线、插针是实盘风险

## 二、市场结构与交易规则

### 交易所与交易时间

| 市场 | 主要交易所 | 交易时间（当地时间） |
|------|-----------|-------------------|
| A 股 | 上交所、深交所、北交所 | 9:30–11:30，13:00–15:00 |
| A 股期货 | 中金所、上期所、大商所、郑商所 | 白天 + 夜盘（各品种不同） |
| 港股 | 港交所 | 9:30–16:00（午休 12:00–13:00） |
| 美股 | NYSE、NASDAQ | 9:30–16:00（盘前盘后各约 4 小时） |
| 加密货币 | Binance、OKX 等 | 7×24 |

### 撮合机制

交易所使用**价格优先、时间优先**原则撮合订单：

- **集合竞价**：开盘前（A 股 9:15–9:25）、收盘前（最后几分钟），一次性产生开盘/收盘价
- **连续竞价**：交易时段内，每进来一个订单立即尝试撮合

> **量化影响**：1 分钟 K 线在集合竞价时段的第一根需要特殊处理，有些数据源会单独标记集合竞价数据。

### 涨跌停（A 股）

| 类型 | 涨跌停幅度 |
|------|-----------|
| 主板 | ±10% |
| 科创板/创业板 | ±20% |
| ST 股票 | ±5% |
| 新股上市首日 | 不设涨跌幅限制（有临时停牌） |

遇到涨跌停时，你的限价单无法成交，市价单可能以严重偏离预期的价格成交。**回测中最好根据涨跌停状态调整买卖信号的可执行性**。

### T+0 与 T+1

| 规则 | 说明 | 量化影响 |
|------|------|----------|
| T+0 | 当天买入可当天卖出（期货、期权、美股、港股、可转债） | 策略可以日内进出 |
| T+1 | 当天买入次日方可卖出（A 股） | 日内信号只能隔夜平仓 |

## 三、订单类型

| 订单类型 | 含义 | 使用场景 |
|----------|------|----------|
| 市价单 | 按当前最优价立即成交 | 追求成交速度，不关心价格 |
| 限价单 | 指定价格，达到或更优才成交 | 控制成本，不保证成交 |
| 止损单 | 价格触及止损位后转为市价单 | 自动止住亏损 |
| 止损限价单 | 价格触及止损位后挂限价单 | 比止损单多了价格保护 |
| 冰山订单 | 只暴露部分数量，隐藏总量 | 大额交易不惊动市场 |

> **回测建议**：中低频策略用限价单 + 滑点估计即可模拟真实成交，无需精确模拟订单簿。

## 四、除权除息与复权 ★

> 这是回测数据处理中最基础也最容易犯错的一环。

### 除权除息是什么

- **除权**：公司送股、转增股本时，股价按比例下调（如 10 送 10，股价腰斩）
- **除息**：公司派发现金红利时，股价扣除对应金额
- 除权除息后，股价出现**非交易性跳空**，如果不处理，任何基于价格的策略都会产生错误信号

### 复权方式

| 方式 | 做法 | 适用场景 |
|------|------|----------|
| **[前复权](../glossary/basics/forward-adjust.md)** | 调整除权前的价格，使历史价格与当前价格连续 | 回测和策略研究最常用 |
| **后复权** | 调整除权后的价格，使价格真实反映历史累计收益 | 查看长期真实涨幅 |
| **不复权** | 保持原始数据不变 | 除权除息日套利策略 |

### Python 中的前复权处理

```python
import pandas as pd

# df 包含 columns: ['open', 'high', 'low', 'close', 'volume', 'adj_factor']
def apply_forward_adjust(df: pd.DataFrame) -> pd.DataFrame:
    """前复权：以最后一个交易日为基准"""
    df = df.sort_index()
    adj = df['adj_factor'] / df['adj_factor'].iloc[-1]
    for col in ['open', 'high', 'low', 'close']:
        df[col] = df[col] * adj
    df['volume'] = df['volume'] / adj    # 成交量反向调整
    return df
```

大多数数据源（AKShare、Tushare）已经提供复权数据，但理解原理很重要：**当回测信号恰好出现在除权除息日附近时，很可能是没正确处理复权导致的假信号。**

## 五、交易成本

| 成本类型 | A 股 | 期货 | 说明 |
|----------|------|------|------|
| 佣金 | 万 1–万 3 | 按手数或比例 | 量化量大可压到很低 |
| 印花税 | 卖出收 0.05% | 无 | 单向收取 |
| 过户费 | 万 0.1 | 无 | 沪深两市收取 |
| 滑点 | 模拟估计 | 模拟估计 | 预期价与实际成交价之差 |
| 冲击成本 | 模拟估计 | 模拟估计 | 大单拉高/压低价格 |

**回测中必须包含交易成本**。一个简单的保守估计：

```python
def estimate_cost(price: float, shares: int, is_sell: bool = False) -> float:
    commission = price * shares * 0.0003          # 万 3
    stamp = price * shares * 0.0005 if is_sell else 0  # 卖出印花税
    slippage = price * shares * 0.001             # 1‰ 滑点
    return commission + stamp + slippage
```

> 建议用 1‰–2‰ 单边成本做保守估计。策略在这个成本下仍盈利，才算通过初步筛选。

## 六、关键财务指标

选股策略（多因子模型）的常用原料：

| 指标 | 简化公式 | 含义 |
|------|---------|------|
| PE（市盈率） | 股价 ÷ 每股收益 | 估值高低，高分位可能高估 |
| PB（市净率） | 股价 ÷ 每股净资产 | 重资产行业参考价值高 |
| ROE（净资产收益率）| 净利润 ÷ 净资产 | 盈利能力的核心指标 |
| ROA（总资产收益率）| 净利润 ÷ 总资产 | 综合资产利用效率 |
| 市值 | 股价 × 总股本 | 大小盘风格区分 |
| 股息率 | 每股分红 ÷ 股价 | 现金回报能力 |
| 换手率 | 成交量 ÷ 流通股本 | 流动性指标 |

**不要只看单一指标**。PE 低不一定是价值股（可能基本面恶化），市值小的公司 ROE 可能不稳定。多因子模型就是把这些指标组合起来互相补充。

## 七、市场微观结构

> 中低频策略（日频以上）了解一下即可，无需深究。

- **买卖价差**：卖一价与买一价的差值，是你隐性的交易成本。高流动性股票价差只有 1 分钱。
- **订单簿深度**：每个价格档位的挂单量，深度越大，大单冲击成本越小。
- **成交量与持仓量**（期货）：
  - 成交量放大 + 价格突破 → 突破可信度高
  - 成交量萎缩 + 价格上涨 → 动力不足，可能反转
  - 持仓量增 + 价格上涨 = 多头控盘；持仓量减 + 价格上涨 = 空头平仓

## 八、常见量化错误

1. **前视偏差**：把未来数据用于生成信号。**修正**：对齐时间戳，确保信号生成时只用历史数据。
2. **幸存者偏差**：只回测存活股票，忽略已退市的。**修正**：使用全样本数据。
3. **忽略交易成本**：理想化成交，低估实际损耗。**修正**：包含佣金、滑点、冲击成本。
4. **过拟合**：对样本反复调参直到结果好看。**修正**：样本外测试、交叉验证、参数敏感性分析。
