# 基础知识

量化交易的起点。理解金融市场运作规则，才能正确地处理数据、设计策略和控制风险。

## 本篇目标

- 理清股票、期货、期权等品种的核心差异
- 掌握交易规则与撮合机制
- 理解除权除息、复权、交易成本等对回测的关键影响
- 了解市场微观结构（订单簿、买卖价差）
- 熟练用 Python 获取并检查行情数据
