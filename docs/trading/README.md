# 实盘与风控

量化研究的第四层。把策略从回测推向实盘，核心不是怎么赚钱，而是怎么**不亏光**。

## 本篇目标

- 理解凯利公式与仓位管理的关系
- 建立多层级风控体系（单笔 → 日级 → 账户级）
- 掌握实盘接口选型与部署流程
- 学会灰度上线：模拟盘 → 小资金 → 加仓
