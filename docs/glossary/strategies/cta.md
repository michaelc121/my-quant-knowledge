# CTA（商品交易顾问）

> **来源文档**：[策略分类框架与案例](../../strategies/strategy-taxonomy.md)
> **前置要求**：理解趋势跟踪的基本概念
> **更新日期**：2026-07-03

## 一句话理解

CTA 最初叫"商品交易顾问"，但今天它其实是一种**跨资产类别的系统性趋势跟踪策略**——在期货市场（商品、股指、债券、外汇）上捕捉大趋势，做多做空均可。

## 名字的由来

CTA = Commodity Trading Advisor。这个词诞生于 1970 年代的美国：

- 当时美国监管机构（CFTC）要求管理商品期货账户的人必须注册为 CTA
- 早期 CTA 确实只交易商品期货（黄金、大豆、原油等）
- 随着市场发展，CTA 开始交易股指期货、债券期货、外汇期货
- **名字没变，但交易范围早就超出"商品"了**

今天"CTA 策略"和"管理期货"（Managed Futures）基本是同义词。

## CTA 策略的核心特征

### 1. 趋势跟踪是绝对主力

绝大多数 CTA（约 70-80%）以趋势跟踪为主：

```text
CTA 趋势跟踪的典型框架：
- 覆盖 4-6 个资产类别：股指、债券、商品、外汇、能源、农产品
- 每个类别 3-10 个品种，总共 20-50 个期货合约
- 每个品种用相同或相似的策略逻辑（如双均线、ATR 通道突破）
- 组合层面的风控管理整体风险暴露
```

### 2. 多品种、多周期

| 类别 | 周期 | 典型持仓 |
|------|------|---------|
| 短线 CTA（高频） | 分钟-小时级 | 几小时到几天 |
| 中频 CTA | 日级 | 几天到几周 |
| 长线 CTA（经典海龟式） | 周-月级 | 几周到几个月 |

著名的元盛（Winton）、AHL、Renaissance（部分产品）都属于 CTA 范畴。

### 3. 做多和做空都可以

CTA 不限于做多——趋势向下时就做空。这是 CTA 和传统股票多头的本质区别。

```
CTA 在 2008 年金融危机中表现很好 → 股票大跌，但 CTA 可以持有做空头寸
CTA 在 2020 年 Q1 新冠暴跌中也表现很好 → 同样的逻辑
但这不意味着 CTA "对冲"了股票风险 → CTA 的趋势跟踪在震荡市中也会亏钱
```

## CTA 与量化交易的交叉

CTA 策略天然适合量化：

```python
# 一个极简的 CTA 信号框架
def cta_trend_signal(price, ma_fast=20, ma_slow=60):
    """
    CTA 常用的趋势信号：双均线交叉
    和普通均线策略没有本质区别，但这是 CTA 的用法不同之处：
    - 同上信号用在同一篮子品种上
    - 不做个股筛选（那是因子选股的事）
    - 仓位通过波动率（ATR）统一管理
    """
    signal = ma_fast - ma_slow
    
    # 标准化为 -1 到 1 的信号强度
    # 正 = 多头，负 = 空头
    return signal / signal.std()
```

CTA 的系统化程度很高：从信号生成、仓位计算到风控执行，全部代码化。

## CTA 的适用条件

CTA 策略需要特定的市场环境才能赚钱：

| 市场环境 | CTA 表现 |
|---------|---------|
| 强趋势（单边牛市/熊市） | 大赚 |
| 趋势反转（牛转熊、熊转牛） | 先亏后赚（趋势确认前有回撤） |
| 震荡（窄幅波动） | 持续亏钱（反复被假突破打脸） |
| 高波动率 | 净仓位自动降低（ATR 控制） |

**CTA 最怕的不是大跌，是长时间的窄幅震荡**——即"趋势跟踪的死亡区"。

## CTA 在 A 股的对应

A 股市场没有成熟的 CTA 生态，但以下几个方向可以类比或直接实践：

1. **商品期货 CTA**：通过 CTP 接口交易国内商品期货（螺纹钢、铁矿石、原油、甲醇等）。这是最接近海外 CTA 的中国市场。
2. **股指期货**：IF（沪深 300 期货）、IC（中证 500 期货）。可以做多空双向的趋势跟踪。
3. **ETF 轮动**：不同品种间轮动（50ETF/创业板ETF/商品ETF/债券ETF），虽然不是严格意义的 CTA，但逻辑类似。

## 常见误区

1. **"CTA 就是投资商品"**：错。CTA 是"系统性趋势跟踪"的代名词，交易范围覆盖股、债、商、汇四大类。
2. **"CTA 是对冲基金"**：部分 CTA 是对冲基金，但并非所有对冲基金都是 CTA。CTA 特指以期货为交易工具、以趋势跟踪为主要策略的基金。
3. **"CTA 能赚大钱"**：CTA 的长期夏普比率通常在 0.3-0.6 之间，不算高。它们的价值在于**股票大跌时的低相关性**，而非绝对值回报。
4. **"散户也能做 CTA"**：CTA 的资金门槛确实低于多因子选股（不需要 500 只股票的数据），但期货的杠杆特性要求严格的风控。小资金做单品种 CTA 的风险比做股票大得多。
