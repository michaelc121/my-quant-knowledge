# my-quant-knowledge

一份结构化的量化交易知识库，用 Markdown 文档承载核心概念、策略原理、代码片段和实操笔记，帮助新手系统性地从零搭建自己的量化认知框架。

## 快速开始

```bash
git clone <this-repo>
cd my-quant-knowledge

# 按学习路线图逐模块阅读
# 起点：docs/basics/market-finance.md
# 实践：notebooks/ 目录下的三个项目
```

## 三个实践项目

| 项目 | 目标 | 建议时间 |
|------|------|---------|
| 项目一：双均线策略 | 从数据到回测报告，端到端跑通 | 第 6 周 |
| 项目二：多因子选股 | 构建 5 个因子，合成得分，分层回测 | 第 8 周 |
| 项目三：实盘模拟盘 | 在模拟盘上跑策略，观察真实市场行为 | 第 10 周 |

## 文档规范

- 每篇 `.md` 文档都有前置说明：内容概述、前置知识、更新时间
- 代码块可复制运行，不省略关键步骤
- 文档之间通过交叉引用形成知识网络，核心概念可通过 [术语表](docs/glossary/README.md) 深入下钻
- 参考外部资料注明来源链接
